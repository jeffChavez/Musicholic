Performance

The performance numbers below are given only as reference data points and are not guaranteed, as many factors come into play in real systems, such as other competing processes, hardware limitations and the set of data being tested with. Performance measurements can have large variations depending on the platform used. The measurements below were done on a Beagle Board xM unless otherwise noted.

CD Lookup Performance

Local CD recognition through a CD TOCTable of Contents. An area on CDs, DVDs, and Blu-ray discs that describes the unique track layout of the disc. is relatively quick with local metadata available. Performance numbers for a local CD recognition can be well under a second. Online CD recognition is slower due to network latency and data transfer time. The following CD Lookup results were measured on the Beagle Board xM against the embedded North America (NA-0010) regional Gracenote Database (GDB) for local lookups. These measurements are represented as an average and max time for the 25K CD TOCs tested.

Lookup Type Average Time (sec) Max Time (sec)
Local CD Lookup 0.01 0.22
Local CD Lookup + Image 0.235 0.77
Online CD Lookup 0.3 12.4
Online CD Lookup + Image 0.8 30.24
