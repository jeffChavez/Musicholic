Detailed Description

Configures and enables GNSDK logging including registering custom logging packages and writing your own logging message to the GNSDK log

Write a message to the custom registered log

Parameters
line[in] Source line number of this call
fileName[in] Source file name of this call
customPackageId[in] Custom Package ID of application making call
messageType[in] Logging message type
format[in] Log message format

Remarks:

This function enables an application to write a logging message to the GNSDK logging system. If a logging events delegate is provided that handles the given package ID and enabled log filters, then this message is written to those log(s). The customer Package ID must be one previously registered by the application.
